# Map

The map should be very simple

## Logical implementation

Should be custom implementation, do not use Godot tilemaps

Each tile type has only two parameters: id (string) and passable (bool)

## Visual implementation

Straight forward

## Tiles

5 tiles:

- Plains (passable)
- Desert (passable)
- Tundra (passable)
- Water (inpassable)
- Mountains (inpassable)
